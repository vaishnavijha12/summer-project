# 📊 Algorithm Visualizer  
### Interactive Sorting & Searching Visualization Tool  

---

## 📌 Overview  
This project is an interactive web-based tool that visualizes fundamental algorithms like sorting and searching. It helps users understand how algorithms work step-by-step through animations.

---

## ✨ Features  
- 🔍 Searching Algorithms Visualization  
- 🔃 Sorting Algorithms Visualization  
- 📊 Step-by-step animation  
- ⚡ Real-time updates  
- 🎨 Clean and interactive UI  

---

## 🧠 Algorithms Included  

### 🔹 Sorting  
- Bubble Sort  
- Selection Sort  
- Insertion Sort  

### 🔹 Searching  
- Linear Search  
- Binary Search  

---
